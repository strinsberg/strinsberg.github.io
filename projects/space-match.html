You can also try the game online, though it is not made to be a web app. The UI is a bit awkward and high scores only last for the session. Menu items can be activated by pressing the key in brackets. For example, at the main menu press n to start a new game or press h to view the help screen. However, during the game the mouse is used to select blocks and select which block to swap them with.
